2.1 The Statistical Computing Language

R
R is a programming language used for data manipulation, statistics, graphics, and reproducible analysis.


R is useful because it can:

  • perform calculations on numbers, vectors, arrays, and matrices
  • generate high-quality graphs
  • use built-in functions and user-contributed packages
  • save analysis steps in scripts that can be reused later

Why Use R?

Point-and-click software
Often easier at first, but steps may be harder to document and repeat.

R scripts
Each step is written down, so the analysis is easier to check, repeat, and modify.


Main idea:
R makes statistical work reproducible, transparent, and easier to automate.

2.2 Installing R and RStudio

Before working with R code, students need two things:

R
The programming language that performs the calculations.

RStudio
The interface used to write scripts, run code, view files, and inspect output.


Think of R as the engine, and RStudio as the dashboard used to control it.

2.2.1 Installing R

Step 1
Go to the official R project website: cran.r-project.org


Choose your operating system:
  • Windows: Download R for Windows → base → download the installer
  • macOS: Download R for macOS → choose the correct .pkg file
  • Linux: Follow the distribution-specific instructions


After installation, R can be accessed from a terminal or through RStudio.

Installing R: Teaching Note

Important:
Students should install R before installing RStudio, because RStudio needs an R installation to run code.


The ebook screenshot shows the CRAN page where students choose their operating system and download the correct installer.

2.2.2 Installing RStudio

RStudio
RStudio is an integrated development environment designed for working with R.


It provides:

  • a script editor
  • an R console
  • an environment and history viewer
  • a file browser
  • plots, packages, help, viewer, and presentation panes

RStudio Workflow

Basic workflow:
  1. Open RStudio.
  2. Create a new script.
  3. Write code in the Source pane.
  4. Run the code.
  5. Read the output in the Console or Plots pane.


Shortcut:
Use Cmd + Enter (Mac) or Ctrl + Enter (Windows) to run the current line of code.

The Four Main RStudio Panes

Pane Purpose
Source Write and edit scripts or Quarto files
Console Run commands and view immediate output
Environment / History View saved objects and past commands
Files / Plots / Packages / Help / Viewer Manage files, plots, packages, and documentation

2.3.2 Basic R Data Structures

The primary data structures introduced in the ebook are:

  • Vectors
  • Lists
  • Matrices
  • Arrays
  • Factors
  • Data frames


Main idea:
Different data structures organize information in different ways.

Vectors

Vector
A one-dimensional sequence of values of the same type.



Vectors are the basic building blocks for many R operations.

Lists

List
A flexible structure that can store elements of different types and sizes.


Matrices

Matrix
A two-dimensional data structure with rows and columns. All elements must be the same type.


Matrix Operations


The operator %*% is used for matrix multiplication. t() transposes a matrix.

Arrays

Array
A multi-dimensional structure that can have more than two dimensions.


Factors

Factor
A data structure used to represent categorical variables.



Factors are useful for categorical data such as gender, education level, or survey responses.

Data Frames

Data frame
A table-like structure where columns are variables and rows are observations.


Data Structure Summary

Structure Shape Best Used For
Vector One dimension Values of the same type
List Flexible Mixed types or grouped information
Matrix Two dimensions Same-type numerical tables
Array Multiple dimensions Higher-dimensional data
Factor One dimension Categorical variables
Data frame Rows and columns Real datasets

2.3.3 Using Basic Functions in R

R has many built-in functions that help calculate, transform, and summarize data.

Function Logic

General form
Most R functions use the structure function_name(input).


For example, mean(x) tells R to apply the function mean to the object x.


Common mistake:
Forgetting brackets, commas, or quotation marks will cause R to return an error.

2.3.5 Installing and Loading Packages

Package
A package is a collection of extra functions, data, and tools that extend what R can do.


Install a package once: 
install.packages("ggplot2")
Load it every new R session: 
library(ggplot2)

Package Reminder

Important:
You install a package only once, but you must load it with library() every time you restart RStudio.


Chapter 2 Summary

  • R is used for statistics, graphics, data manipulation, and reproducible analysis
  • RStudio is the main interface for writing and running R code
  • R can be used as a calculator for arithmetic and mathematical functions
  • Vectors, lists, matrices, arrays, factors, and data frames each store data differently
  • Functions take inputs and produce outputs using the form function_name(input)
  • Histograms display the distribution of a numerical variable
  • Packages extend R’s functionality and must be loaded each session with library()
