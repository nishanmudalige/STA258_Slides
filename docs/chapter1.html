1.1 Foundations

Definition 1.1: Statistics
Statistics is the science of collecting, classifying, summarizing, analyzing, and interpreting data.


Statistics helps researchers use data to support conclusions about a larger group.

Main idea: Statistics helps us move from data to evidence, and from evidence to decisions.

Emphasize that statistics is not just formulas — it is about learning from data when there is uncertainty.

Population and Sample

Population
The entire group of individuals, items, or measurements we want to study.

Sample
A smaller subset selected from the population.


Key idea: We usually study the sample because studying the entire population is too difficult, expensive, or impossible.

Example: Population and Sample

Population: All students at the University of Toronto.

Sample: 50 randomly selected students from the University of Toronto.


Main idea:
We study the sample to learn about the population.

Make sure students understand that the sample is not the final target — the population is.

Unit, Observation, Variable

Unit
The smallest entity from which data are collected.


Observation
The collection of measured values for one unit.


Variable
A characteristic that can be measured or recorded.

Example: Unit, Observation, Variable

If we record a student’s age, height, and weight:

  • The student is the unit
  • The full row of recorded values is the observation
  • Age, height, and weight are variables

Quick check:
If a dataset has 100 students and 4 measured characteristics per student, how many observations are there?

Population Unit vs Sample Unit

Population unit
One member of the full population.


If the population is all UTM students, one UTM student is a population unit.

Sample unit
One member of the selected sample.


If the sample is 80 selected UTM students, one selected student is a sample unit.

Dataset Language

Term Meaning Example
Unit Who or what data are collected from One student
Observation Full row of data for one unit Age, height, weight for one student
Variable Characteristic being recorded Age


This table is useful when students first learn how to read datasets.

Parameter vs Statistic

Parameter
A numerical quantity that describes a population.

Statistic
A numerical quantity that describes a sample.


Memory trick:
Parameter → Population   |   Statistic → Sample

Parameter vs Statistic: Examples

Parameters (population-level):
  • Population mean, \(\mu\)
  • Population standard deviation, \(\sigma\)
  • Population proportion, \(p\)


Statistics (sample-level):
  • Sample mean, \(\bar{x}\)
  • Sample standard deviation, \(s\)
  • Sample proportion, \(\hat{p}\)

Students often mix up \(p\) and \(\hat{p}\). Stress that \(p\) is population-level and \(\hat{p}\) is sample-level.

Parameter vs Statistic: Summary Table

Population Quantity Sample Quantity Meaning
\(\mu\) \(\bar{x}\) Mean
\(\sigma\) \(s\) Standard deviation
\(p\) \(\hat{p}\) Proportion


We use sample statistics like \(\bar{x}\) or \(\hat{p}\) to estimate population parameters like \(\mu\) or \(p\).

Descriptive vs Inferential Statistics

Descriptive statistics
Methods used to analyze, summarize, and represent data.


Inferential statistics
Using sample information to make conclusions about a larger population.

Descriptive vs Inferential: Examples

Descriptive

The average mark in a class of 80 students is 72%.

Inferential

Based on 80 students, we estimate the average mark for all STA258 students.

1.2 Types of Data

Quantitative data
Data that can be measured numerically.


Examples: height, weight, age, temperature


Question:
Does every number automatically mean the variable is quantitative?

Qualitative Data

Qualitative data
Data that cannot be measured numerically — instead, it falls into categories.


Examples: favourite colour, eye colour, ice-cream flavour

Quantitative vs Qualitative

Data Type Reason
Height Quantitative Measured numerically
Number of courses Quantitative Counted numerically
Favourite colour Qualitative Falls into a category
Student number Qualitative The number is only an identifier

The student number example is useful — it shows that numbers are not always quantitative.

Discrete and Continuous Data

Discrete data
Quantitative data that are countable.


Examples: number of siblings, number of defects

Continuous data
Quantitative data measured on a scale.


Examples: height, time, temperature

Quick Check: Types of Data

Classify each variable:

  1. Number of courses a student is taking
  2. Student’s major
  3. Time spent studying per week
  4. Whether a student lives on campus
Answers:
  1. Quantitative, discrete
  2. Qualitative
  3. Quantitative, continuous
  4. Qualitative

1.3 Introduction to Inferential Statistics

Sample → Statistic → Estimate population parameter


Statistical inference uses sample data to estimate population values and measure uncertainty.

\[ \text{Sample} \rightarrow \text{Statistic} \rightarrow \text{Parameter} \]

Big Picture

We usually cannot measure the whole population.

So we collect a sample, calculate a statistic, and use it to estimate the unknown parameter.


Important:
Because samples vary, inference must always include a measure of uncertainty.

Main Inference Tools

Estimation
Using sample data to guess the value of a population parameter.

Confidence intervals
Giving a plausible range of values for a population parameter.

Hypothesis testing
Checking whether sample data support a specific claim.

Pause before revealing the answers. This is a good check for whether students understand sample vs population and statistic vs parameter.

Chapter 1 Summary

  • Statistics is the science of learning from data
  • A sample is used to study a population
  • Parameters describe populations; statistics describe samples
  • Data can be quantitative or qualitative
  • Quantitative data can be discrete or continuous
  • Inference uses samples to draw population-level conclusions
